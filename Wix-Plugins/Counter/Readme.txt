## 📌 Overview

This code creates a smooth and visually engaging <strong>counter animation</strong>, which increases a number from a starting value to a target value over a defined duration.

You can use it to display:

- 📆 Years of Experience
- ✅ Accuracy Rate
- 🌟 Positive Reviews
- 🤝 Trusted Partners
